# DashCartola
Dashboard com informações do cartola e sofascore para auxiliar cartoleiros a escalarem seu time

O código info_sofascore é utilizado para raspar os dados do SofaScore. Bibliotecas convencionais como beautifulsoup não conseguem extrair as informações e, portanto, precisamos de apelar para o Selenium.

O código main_brenomestre (uma clara alusão ao Gato Mestre, dashboard da TV Globo que apenas assinantes têm acesso) é utlizado para criar e adicionar novas informações à todas as tabelas utilizadas.

O código app é o dashboard em si. Ele é programado de acordo com a documentação do Streamlit.
